    How CIPHERLOCK reads a password

    1. Security strength measures resistance to guessing: length, character variety, entropy, and known weak patterns.
    2. Cognitive load measures how much mental effort the password takes to recall — randomness that's hard to type twice in a row.
    3. Memorability rewards structure a human mind can hold onto: pronounceable chunks, real words, and separators — without falling back on predictable sequences.
